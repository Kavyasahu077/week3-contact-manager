# Contact Management System

## Project Description

Contact Management System is a Python project developed using functions and dictionaries.

The program allows users to add, search, update, delete, and view contacts using a menu-driven interface.

This project helped me understand dictionaries, file handling, functions, input validation, and JSON storage in Python.

---

# Features

- Add new contacts
- Search contacts using partial matching
- Update contact details
- Delete contacts
- View all contacts
- Save contacts automatically using JSON
- View contact statistics
- User-friendly menu system
